# Documentação do repositório "exemples_dart"

Visão geral

- Repositório com exemplos em Dart/Flutter focados em navegação (auto_route), rotas declarativas, fontes customizadas e organização modular.
- Cada exemplo está isolado em subdiretórios dentro de `lib/`.

Estrutura

- assets/fonts/       -> fontes customizadas usadas nos exemplos
- lib/                -> código dos exemplos (rotas/mobile, rotas/declarative, routes_simples, test_fonts, etc.)
- router_config.dart  -> configuração central de rotas (auto_route)

Como executar

1. flutter pub get
2. flutter run

Contribuição

- Siga as convenções do Effective Dart.
- Nome de arquivos: lowercase_with_underscores.
- Mantenha cada exemplo modular e isolado.
